Introdução ao CSS

Sintaxe

   regra       declaração

      ↓                    ↓

Seletor{ propriedade : valor ;}

  • Seletor: Determina quais elementos HTML serão estilizados.
  • Propriedade: Especifica o aspecto do estilo que você deseja modificar (por exemplo, cor, tamanho de fonte, margem, etc.).
  • Valor: Define o valor que será atribuído à propriedade, como a cor, o tamanho, o tipo de fonte, etc.
  • Declaração: É a combinação de uma propriedade e seu valor. As declarações são terminadas com um ponto e vírgula (;).

P.ex: h1{ color : red ;}

Como vincular os arquivos

Para vincular um arquivo CSS externo a um documento HTML, você precisa usar a tag <link> dentro da seção <head> do seu arquivo HTML.

Essa tag conecta o arquivo CSS ao HTML, permitindo que o CSS estilize o conteúdo da página.

No arquivo HTML, adicione a seguinte linha no <head>:

  • <link> rel="stylesheet" href="arquivo.css">

Principais propriedades do CSS

Tipos de fontes

  • Font-family: Define a família da fonte utilizada.
  • Font-size: Define a propriedades de estilos que podem ser: normal, italic ou oblique.
  • Font-style: Define o tamanho da fonte.

Text-transform

Controla os efeitos de capitalização do texto. Com ela, podemos definir a caixa das letras (minúsculas e maiusculas). Os valores possíveis são:

  • capitalize:Capitalize transforma o primeiro caracter de cada palavra em maiúscula.
  • uppercase:Transforma todas as letras de todas as palavras em maiúsculas
  • lowercase:Transforma todas as letras de todas as palavras em minúsculas
  • none:Cancela algum valor que tenha sido herdado

Seletores: classe e ID

Class

  • Começa com um . (caractere ponto). Ele selecionará tudo no documento com essa classe aplicada a ele. Todos os elementos que têm a classe aplicada são destacados

Id

  • começa com um # (caractere) em vez de um (.) ponto, mas é usado da mesma forma que um seletor de classe. No entanto, ID pode ser usado apenas uma vez por página, e os elementos podem ter apenas um único id valor aplicado a eles . Ele pode selecionar um elemento que tenha o id conjunto nele, e você pode preceder o ID com um seletor de tipo para direcionar o elemento somente se o elemento e o ID corresponderem

Box Model

O Box Model começa com o conteúdo como ponto de partida porque ele é a parte essencial e mais interna do elemento, onde o conteúdo real, como texto ou imagens, é colocado. Esse foco no conteúdo facilita o controle sobre como ele será exibido e como será ajustado para comportar o layout ao redor dele. A borda e outros elementos do Box Model (padding e margem) são camadas externas adicionadas ao conteúdo para controlar a aparência e o espaçamento.

Content (Conteúdo)

  • Esta é a área onde o conteúdo real do elemento é colocado (texto, imagem, etc.).
  • A largura e altura definidas de um elemento geralmente se referem a essa área.

Padding (Preenchimento)

  • Espaço entre o conteúdo do elemento e sua borda.
  • Padding aumenta o tamanho da caixa total, pois empurra a borda para fora
  • Pode ser definido individualmente para cada lado (topo, direita, baixo, esquerda).

Border (Borda)

  • Envolve o conteúdo e o padding, criando uma borda ao redor do elemento.
  • Pode ser personalizada em termos de largura (border-width), estilo (border-style, como sólida, tracejada, etc.) e cor (border-color).

Margin (Margem)

  • Espaço entre a borda do elemento e os elementos vizinhos.
  • Pode ser usado para criar espaçamento externo e afastar elementos uns dos outros.
  • Também pode ser definido individualmente para cada lado do elemento.
